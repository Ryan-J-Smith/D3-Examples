# Plotting Random Normal Data This is an initial simple demonstration in visualizing bivariate normal data. For each data point the coordinates are assigned by drawing two independent random samples from a standard normal distribution. This code borrows heavily from Mike Bostock's example [here](http://bl.ocks.org/mbostock/3680957). <div id="d3Code"></div> ## D3 Code var w = 500; var h = 500; var std = 1; var numPoints = 1000; //Generate random data var randomX = d3.random.normal(0, std); var randomY = d3.random.normal(0, std); var dataset = d3.range(numPoints).map(function() { return [ randomX(), randomY() ]; }); //Define scale var xScale = d3.scale.linear() .domain([-10*std, 10*std]) .range([0, w]); var yScale = d3.scale.linear() .domain([10*std, -10*std]) .range([0, h]); //Create SVG element var svg = d3.select("div#d3Code").append("svg") .attr("style", "outline: thin solid black;") .attr("width", w) .attr("height", h); //Draw data on svg element svg.selectAll("circle") .data(dataset) .enter() .append("circle") .attr("cx", function(d) { return xScale(d[0]); }) .attr("cy", function(d) { return yScale(d[1]); }) .attr("r", 2.5); --- <div class="container"> <div class="row"> <span class="pull-left"> <a class="btn btn-primary" href="http://Ryan-J-Smith.github.io/D3-Examples">Back to Examples</a></span> <span class="pull-right">By [Ryan J.
